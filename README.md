# MoneyTrackerProject - เว็บแอปบันทึกรายรับรายจ่าย

เว็บแอปบันทึกรายรับรายจ่ายสำหรับนักศึกษามหาวิทยาลัย พัฒนาด้วย HTML, CSS, และ JavaScript อย่างง่าย

## คุณสมบัติหลัก

### ✅ ฟีเจอร์ที่พร้อมใช้งาน

1. **บันทึกรายรับ/รายจ่าย**
   - เพิ่มรายการรายรับ/รายจ่ายพร้อมโน้ต
   - เลือกบัญชีและหมวดหมู่
   - ระบุวันที่และเวลา

2. **จัดการหลายบัญชี**
   - เพิ่ม/แก้ไข/ลบบัญชี (เช่น ใช้จ่าย, ออม, รายได้พิเศษ)
   - แสดงยอดคงเหลือของแต่ละบัญชี
   - โอนเงินระหว่างบัญชี

3. **หมวดหมู่และงบประมาณ**
   - ตั้งงบประมาณรายเดือนต่อหมวด
   - แสดงแถบความคืบหน้าการใช้งบ
   - แจ้งเตือนเมื่อใช้งบเกิน 80% และ 100%

4. **เป้าหมายการออม**
   - ตั้งเป้าหมายการออมพร้อมวันสิ้นสุด
   - แสดงแถบความคืบหน้า
   - เชื่อมโยงกับบัญชี

5. **กราฟและสถิติ**
   - กราฟแสดงสัดส่วนรายจ่ายตามหมวด
   - กราฟเส้นแนวโน้มรายสัปดาห์
   - กราฟเปรียบเทียบรายรับ/รายจ่ายรายเดือน

6. **ระบบเกมและชาเลนจ์**
   - ชาเลนจ์รายสัปดาห์ (เช่น ไม่ซื้อกาแฟ 5 วัน, ออมวันละ 20 บาท)
   - มินิเกม: ดวงไฟการออม (ไฟจะสว่างเมื่อออมเงินตามเป้าหมายทุกวัน)

7. **การยืมและให้ยืมเงิน**
   - บันทึกการยืม/ให้ยืม
   - ระบุวันที่ครบกำหนด
   - แสดงสถานะ (รอชำระ, ชำระครบ, เกินกำหนด)

8. **การส่งออก/นำเข้าข้อมูล**
   - ส่งออกข้อมูลเป็น CSV
   - นำเข้าข้อมูลจาก CSV

9. **ธีมและความปลอดภัย**
   - โหมด Dark/Light
   - โหมดซ่อนยอดเงิน
   - ข้อมูลเก็บใน localStorage (ทำงานออฟไลน์ได้)

## การติดตั้งและใช้งาน

## วิธีใช้งาน

### เพิ่มรายการรายรับ/รายจ่าย
1. คลิกที่แท็บ "เพิ่ม" ที่ด้านล่าง (ไอคอน +)
2. เลือกประเภท (รายรับ/รายจ่าย)
3. ใส่จำนวนเงิน, เลือกบัญชี, หมวดหมู่, วันที่
4. คลิก "บันทึก"

### จัดการบัญชี
1. ไปที่แท็บ "บัญชี" ที่ด้านล่าง
2. คลิก "เพิ่มบัญชี" เพื่อเพิ่มบัญชีใหม่
3. ใช้ฟอร์ม "โอนเงินระหว่างบัญชี" เพื่อโอนเงิน

### ตั้งงบประมาณ
1. ไปที่แท็บ "ตั้งค่า" → คลิก "งบประมาณ"
2. คลิก "เพิ่มงบประมาณ"
3. เลือกหมวดหมู่และตั้งจำนวนเงิน

### ตั้งเป้าหมายการออม
1. ไปที่แท็บ "ตั้งค่า" → คลิก "เป้าหมายการออม"
2. คลิก "เพิ่มเป้าหมาย"
3. ใส่ชื่อ, จำนวนเป้าหมาย, วันสิ้นสุด, และเลือกบัญชี

### ดูสถิติ
1. ไปที่แท็บ "สถิติ" ที่ด้านล่าง
2. สลับระหว่าง "รายสัปดาห์" และ "รายเดือน"
3. ดูกราฟและข้อมูลวิเคราะห์

### ระบบเกม
1. ไปที่แท็บ "ตั้งค่า" → คลิก "ชาเลนจ์และเกม"
2. ดูชาเลนจ์ที่กำลังทำ
3. เล่นมินิเกม "ดวงไฟการออม" โดยออมเงินตามเป้าหมายทุกวัน

## หมายเหตุ

- ข้อมูลทั้งหมดเก็บใน localStorage ของเบราว์เซอร์
- ข้อมูลจะไม่หายไปเมื่อปิดเบราว์เซอร์
- หากล้างข้อมูลเบราว์เซอร์ ข้อมูลจะหายไป (แนะนำให้ส่งออก CSV เป็นประจำ)
- ฟีเจอร์ล็อกอินและซิงก์ออนไลน์ยังไม่พร้อมใช้งาน (จะพัฒนาต่อในอนาคต)

## การพัฒนาต่อ

ฟีเจอร์ที่ยังไม่ได้พัฒนา:
- ระบบล็อกอิน (Google/อีเมล)
- ซิงก์อัตโนมัติกับเซิร์ฟเวอร์
- การแจ้งเตือนแบบ Push Notification
- การแยกเงินอัตโนมัติตามสัดส่วน 60/30/10
- การค้นหาและกรองข้อมูลขั้นสูง
